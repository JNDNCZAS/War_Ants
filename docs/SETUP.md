# Guía de Configuración e Instalación para el proyecto War_Ants

## Introducción
Este documento proporciona instrucciones sobre cómo configurar y desarrollar el proyecto War_Ants en su máquina.

## Requisitos Previos
Antes de comenzar, asegúrese de tener instalados lo siguiente:
- [Node.js](https://nodejs.org/): 14.x o posterior
- [Git](https://git-scm.com/): Para clonar el repositorio

## Pasos para la Instalación
1. **Clonar el Repositorio**  
   Abra una terminal y ejecute el siguiente comando para clonar el repositorio:
   ```bash
   git clone https://github.com/owner/War_Ants.git
   cd War_Ants
   ```

2. **Instalar Dependencias**  
   Asegúrese de que está en la carpeta del proyecto y ejecute:
   ```bash
   npm install
   ```

3. **Ejecutar el Proyecto**  
   Para iniciar el proyecto, use el siguiente comando:
   ```bash
   npm start
   ```
